# Gestoría de Tráfico

Aplicación web para gestión de expedientes de tráfico: transferencias, canjes, bajas, duplicados, matriculaciones y VMP.

## Funcionalidades

- ✅ Gestión de clientes (personas físicas y jurídicas)
- ✅ Gestión de vehículos
- ✅ Expedientes de transferencias
- ✅ Expedientes de canjes de permisos
- ✅ Expedientes de bajas
- ✅ Expedientes de duplicados
- ✅ Expedientes de VMP (Vehículos de Movilidad Personal)
- ✅ Generación automática de PDFs (contratos, mandatos, MOD-03, etc.)
- ✅ Sincronización con Google Sheets
- ✅ Exportación a Hermes

## Uso

Accede a la aplicación en: https://gestoriajustoguevara-art.github.io/gestoria-trafico/

## Archivos PDF necesarios

La aplicación usa las siguientes plantillas PDF:
- `mandato.pdf` - Mandato profesional
- `contrato.pdf` - Contrato de compraventa
- `CONDUCTORES.pdf` - MOD-03 para trámites de conductores
- `declaracion-baja.pdf` - Declaración de baja de vehículo
- `documento_representacion_juridica.pdf` - Representación de personas jurídicas
- `Mod_30-ES.pdf` - MOD-30 para VMP
- `MODELO_TRANSFERENCIA.pdf` - MOD-02 para transferencias
